YouTube Tags Extractor

Extract title, description and complete tag list from any public YouTube video. Use API key (recommended) or our server proxy.
Tip: For reliable full tag extraction use YouTube Data API v3 (server-side proxy keeps key secret). If you don't have an API key use the proxy or noembed fallback.

How this works (engineer's note)

  1. Preferred: YouTube Data API v3 — reliable, returns full `snippet.tags` array.
  2. Proxy: Server fetches video page and parses `ytInitialPlayerResponse` JSON which often contains `videoDetails.keywords`/`microformat` fields.
  3. Fallback: noembed or OG meta parsing — may not include all tags.

For production use a server-side proxy (see /tools/youtube-tags-extractor/server.js) to keep the API key private and avoid CORS issues.
